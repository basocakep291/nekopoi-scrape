# Nekopoi Scrape

Scrape [Nekopoi](https://nekopoi.care) dengan **Beautifulsoup4** dan **Pypeteer** menggunakan bahasa python

# Routes
`/newEpisodes` = Rute untuk episode terbaru dari nekopoi.
`/search/<anime_name>` = Mencari anime dari nekopoi.
